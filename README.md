## Employee Manager app contains:
* User register 
* User login
* User logout 
* Show all employees
* Show employee profile
* Add new employee
* Edit employee
* Delete employee


## Usage :
### Run project by :

``` vue.js

// install dependencies

1. npm install

// serve with hot reload

2. npm run dev

// build for production

3. npm run build

// Or you can simply run:

> npm i & npm run dev

```

That's it.

## Done :

Now the project is running at `http://localhost:8080` :
